To get access token, you need make request to the authorization service with your credentials and pass it as authorization header

curl -X POST \
  -H "Content-Type: application/x-www-form-urlencoded" \
  -H "Authorization: Basic dXNlcm5hbWU6cGFzc3dvcmQ=" \
  --data 'grant_type=client_credentials' \
  "https://auth.service.com/oauth/token"

in response you are receive object

{
  "access_token": "<ENCRYPTED_JWT_TOKEN>",
  "expires_in": 12345678,
  "token_type": "Bearer"
}

The next step is create request to protected resource, with following authorization header:

Authorization: Bearer <ENCRYPTED_JWT_TOKEN>
